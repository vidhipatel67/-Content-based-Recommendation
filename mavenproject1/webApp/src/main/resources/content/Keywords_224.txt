<p><code><a href="/wiki/Java_Programming/Keywords/private" title="Java Programming/Keywords/private"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">private</font></a></code> is a Java <strong class="selflink">keyword</strong> which declares a member's <a href="/wiki/Java_Programming/Access_Modifiers" title="Java Programming/Access Modifiers" class="mw-redirect">access</a> as private. That is, the member is only visible within the class, not from any other class (including subclasses). The visibility of <code><a href="/wiki/Java_Programming/Keywords/private" title="Java Programming/Keywords/private"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">private</font></a></code> members extends to <a href="/wiki/Java_Programming/Nested_Classes" title="Java Programming/Nested Classes">nested classes</a>.</p>
