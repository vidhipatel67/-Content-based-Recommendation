<p>On a machine using the Windows operating system, one can invoke the Registry Editor utility by typing <code>REGEDIT</code> in the Run dialog. In the window that opens subsequently, if you traverse through the hierarchy <code><a href="//en.wikipedia.org/wiki/HKEY_LOCAL_MACHINE#HKEY_LOCAL_MACHINE" class="extiw" title="w:HKEY LOCAL MACHINE">HKEY_LOCAL_MACHINE</a> &gt; SOFTWARE &gt; JavaSoft &gt; Java Development Kit</code> on the left-hand.</p>
