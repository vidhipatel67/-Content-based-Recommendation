<p>Behind closed doors, a project was initiated in December of 1990, whose aim was to create a programming tool that could render obsolete the C and C++ programming languages. Engineer Patrick Naughton had become extremely frustrated with the state of Sun's C++ and C APIs (Application Programming Interfaces) and tools. While he was considering to move towards <a href="//en.wikipedia.org/wiki/NeXT" class="extiw" title="w:NeXT">NeXT</a>, he was offered a chance to work on new technology and the <i>Stealth Project</i> was started, a secret nobody but he knew.</p>
