<p>Although <a href="//en.wikipedia.org/wiki/Address_space" class="extiw" title="w:Address space">memory spaces</a> have their own addresses — usually a hash number such as <code>0xCAD3</code>, etc. — it is much easier to remember a variable's location in the memory if we can give it a recognizable name. <b>Identifiers</b> are the names we give to our variables. You can name your variable anything like <code>aVariable</code>, <code>someVariable</code>, <code>age</code>, <code>someonesImportantData</code>, etcetera. But notice: none of the names we described here has a space within it. Hence, it is pretty obvious that spaces aren't allowed in variable names.
