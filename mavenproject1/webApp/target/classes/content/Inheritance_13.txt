<p>In this example, the constructor of the <code>Plane</code> class calls the constructor of its superclass <code>Vehicle</code>. You can only use <code><a href="/wiki/Java_Programming/Keywords/super" title="Java Programming/Keywords/super"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">super</font></a></code> to access the members of the superclass inside the child class. If you use it from another class, it accesses the superclass of the other class. This keyword also allows you to explicitly access the members of the superclass, for instance, in the case where there is a method with the same name in your class (overriding, ...).
