<p>Note that C++ JNI code is syntactically slightly cleaner than C JNI code because like Java, C++ uses object method invocation semantics. That means that in C, the <code>env</code> parameter is dereferenced using <code>(*env)-&gt;</code> and <code>env</code> has to be explicitly passed to <code>JNIEnv</code> methods. In C++, the <code>env</code> parameter is dereferenced using <code>env-&gt;</code> and the <code>env</code> parameter is implicitly passed as part of the object method invocation semantics.</p>
