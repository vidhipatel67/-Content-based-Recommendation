<p>Such files can contain only ASCII and Latin-1 characters, but international programmers need a full range of character sets. Text using these characters can appear in properties files and resource bundles only if the non-ASCII and non-Latin-^1 characters are converted into <a href="/wiki/Java_Programming/Syntax/Unicode_Escape_Sequences" title="Java Programming/Syntax/Unicode Escape Sequences" class="mw-redirect">Unicode escape sequences</a> (\uXXXX notation).</p>
