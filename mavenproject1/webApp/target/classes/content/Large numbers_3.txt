<p><br> The integer primitive type with the largest range of value is the <code><a href="/wiki/Java_Programming/Keywords/long" title="Java Programming/Keywords/long"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">long</font></a></code>, from -2<sup>63</sup> to 2<sup>63</sup>-1. If you need greater or lesser values, you have to use the <code>BigInteger</code> class in the package <code>java.math</code>. A <code>BigInteger</code> object can represent <u>any</u> integer (as large as the RAM on the computer can hold) as it is not mapped on a primitive type. Respectively, you need to use the <code>BigDecimal</code> class for great decimal numbers.</p>
