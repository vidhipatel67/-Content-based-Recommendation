<p>However powerful, the predecessors of Java lacked a standard feature to network with other computers, and usually relied on the platforms' intricate networking capabilities. With almost all network protocols being standardized, the creators of Java technology wanted this to be a flagship feature of the language while keeping true to the spirit of earlier advances made towards standardizing <a href="//en.wikipedia.org/wiki/Remote_procedure_call" class="extiw" title="w:Remote procedure call">Remote Procedure Call</a>. Another feature that the Java team focused on was its integration in the World Wide Web and the Internet.</p>
