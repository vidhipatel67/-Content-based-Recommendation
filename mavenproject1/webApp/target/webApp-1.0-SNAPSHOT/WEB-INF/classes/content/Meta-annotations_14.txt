<p>Different annotation types have different purposes. Some are intended for use with the compiler; others are meant to be reflected dynamically at runtime. There's no reason for a compiler annotation to be available at runtime, so the <code>@Retention</code> meta-annotation specifies how long an annotation type should be retained.
