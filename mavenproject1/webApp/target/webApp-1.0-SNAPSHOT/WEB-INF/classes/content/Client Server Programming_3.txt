<p><br> In 1990s, the trend was moving away from <a href="//en.wikipedia.org/wiki/Mainframe_computer" class="extiw" title="w:Mainframe computer">Mainframe computing</a> to Client/Server, as the price of Unix servers dropped. The database access and some business logic were centralized on the <a href="//en.wikipedia.org/wiki/Front-end_and_back-end" class="extiw" title="w:Front-end and back-end">back-end</a> server, collecting data from the user program was installed on the <a href="//en.wikipedia.org/wiki/Front-end_and_back-end" class="extiw" title="w:Front-end and back-end">front-end</a> users' "client" computers. In the Java world there are three main ways the front-end and the back-end could simply communicate.</p>
