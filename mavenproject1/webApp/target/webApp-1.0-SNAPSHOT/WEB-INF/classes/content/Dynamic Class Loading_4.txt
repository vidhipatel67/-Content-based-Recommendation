<p>An easy way to dynamically load a <code>Class</code> is via the <code>java.net.URLClassLoader</code> class. This class can be used to load a <code>Class</code> or a collection of classes that are accessible via a URL. This is very similar to the <code>-classpath</code> parameter in the <code>java</code> executable.
