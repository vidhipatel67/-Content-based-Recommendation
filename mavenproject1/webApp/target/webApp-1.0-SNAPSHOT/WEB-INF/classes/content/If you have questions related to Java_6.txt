<p>CORBA is language independent "Remote Method Invocation" like <a href="/wiki/Java_Programming/Remote_Method_Invocation" title="Java Programming/Remote Method Invocation">RMI</a>. <a href="/wiki/Java_Programming/Remote_Method_Invocation" title="Java Programming/Remote Method Invocation">RMI</a> only works with Java, CORBA can be used with any languages. For more information see: <a href="//en.wikipedia.org/wiki/Common_Object_Request_Broker_Architecture" class="extiw" title="w:Common Object Request Broker Architecture">w:Common Object Request Broker Architecture</a>.</p>
