<p>Java uses <a href="//en.wikipedia.org/wiki/unicode" class="extiw" title="w:unicode">unicode</a>, while C++ uses <a href="//en.wikipedia.org/wiki/ASCII" class="extiw" title="w:ASCII">ASCII</a>. Unicode consists of a repertoire of about 100,000 characters, which is much more than what the ASCII character-set can represent.
